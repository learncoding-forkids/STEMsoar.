Lesson 5: Introduction to Conditionals

Now that we understand variables and how they store information, we are ready to learn a new concept, conditionals! Conditionals help computers decide what to do. Just like you make choices every day, computers do too, but they need clear rules to follow.

A conditional is like saying “If this happens, then do that.” For example, imagine you are outside and you feel raindrops. You might think: If it is snowing, then put on a jacket. You made a decision based on a condition!

Computers do the same thing, but they use code. In coding, a conditional might look like this: if score > 20: say “Nice!” This means: If the score is greater than 20, then the computer should say “Nice!” If the condition is not true, the computer will not do that action. So if the score is 19 it will not say “Nice!”.

Sometimes we also use the word else to tell the computer what to do if the condition is not met. For example: if it is raining: take an umbrella else: wear sunglasses. Conditionals help programs react dynamically to circumstances, whether in games, apps, or complex software architectures.
